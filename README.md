# Estetik Pusula

Kişisel güzellik yol haritanız. AI destekli yüz analizi, kişiselleştirilmiş estetik öneriler, günlük bakım ipuçları ve ilerleme takibi.

## Özellikler

- **AI Yüz Analizi** - 10+ metrikle kapsamlı yüz değerlendirmesi
- **Kişisel Öneriler** - Cerrahi ve medikal estetik önerileri
- **Günlük İpuçları** - Her gün yenilenen bakım tavsiyeleri
- **Rutin Takibi** - Sabah ve akşam bakım rutinlerini kaydedin
- **İlerleme Takibi** - Analizler arası gelişiminizi görün
- **Prosedür Rehberi** - Detaylı işlem bilgileri ve fiyatlar
- **AI Dönüşüm** - İşlem öncesi simülasyon görüntüleri

## Teknoloji

- **Frontend**: React Native + Expo SDK 54
- **Backend**: FastAPI + MongoDB
- **AI**: OpenAI GPT-4o + DALL-E 3
- **Ödeme**: RevenueCat (iOS)
