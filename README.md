# Yüz Atölyem - Kişisel Güzellik Asistanı

AI destekli yüz analizi, cilt günlüğü takibi ve estetik prosedür rehberi.

## Kurulum

### Frontend
```bash
cd frontend
yarn install
npx expo start
```

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --host 0.0.0.0 --port 8001
```

## Özellikler
- AI Yüz Analizi (cerrahi & medikal)
- Cilt Günlüğü (mood, rutin, su, uyku takibi)
- Prosedür Rehberi (fiyatlar, süreler, riskler)
- Günlük Güzellik İpuçları
- Premium AI Dönüşüm Simülasyonu
